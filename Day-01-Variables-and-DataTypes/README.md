# 🐍 Day 01 – Variables & Data Types

> 🎯 **Goal:** Learn how Python stores different kinds of information.

---

# 📦 What is a Variable?

A **variable** is a **named container** used to store information so that we can use it later in our program.

Think of it as giving a **name to a value**.

### 🌍 Real-Life Example

Imagine your school gives every student a locker.

```
Locker No: 101
┌──────────────┐
│  Books 📚    │
└──────────────┘
```

The locker has a **number (name)** and stores **books (value)**.

Similarly in Python,

```
name = "Rahul"
```

- `name` ➜ Variable Name 📦
- `"Rahul"` ➜ Stored Value 📝

---

# 📚 What is a Data Type?

A **data type** tells Python **what kind of information** is stored inside a variable.

Different information needs different data types.

### 🌍 Real-Life Example

Think about filling a school admission form.

| Information | Example | Data Type |
|------------|---------|-----------|
| Name | Rahul | 📝 String |
| Age | 20 | 🔢 Integer |
| Height | 5.8 | 📊 Float |
| Passed? | Yes | ✅ Boolean |

Python stores each value according to its type.
