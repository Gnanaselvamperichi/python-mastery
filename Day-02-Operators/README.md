# 📖 What is an Operator?

An **operator** is a **special symbol** that tells Python to perform an operation on one or more values.

Think of it like giving Python an instruction.

### 🌍 Real-Life Example

Imagine using a calculator.

```
10 + 5 = 15
20 - 8 = 12
6 × 4 = 24
12 ÷ 3 = 4
```

The symbols **+ , - , * , /** are operators.

Python works exactly the same way.

```python
a = 10
b = 5

print(a + b)
```

---

# 🎨 How Operators Work

```
Two Values
    │
    ▼
  Operator
    │
    ▼
   Result

10 + 5
   │
   ▼
  15
```

---

# 📚 Types of Operators

| Operator Type | Purpose |
|--------------|---------|
| ➕ Arithmetic | Perform calculations |
| ⚖️ Comparison | Compare two values |
| 🟰 Assignment | Store or update values |
| 🔗 Logical | Combine conditions |
| 📌 Membership | Check whether a value exists |
| 🆔 Identity | Check if two variables refer to the same object |

---

# 🧠 Memory Trick

```
➕ Calculate ➜ Arithmetic

⚖️ Compare ➜ Comparison

🟰 Assign ➜ Assignment

🔗 Combine ➜ Logical

📌 Search ➜ Membership

🆔 Identify ➜ Identity
```

# 💡 What I Learned Today

- Operators perform actions on values.
- Arithmetic operators are used for calculations.
- Comparison operators return `True` or `False`.
- Assignment operators store values.
- Logical operators combine conditions.
- Membership and Identity operators help compare objects and collections.

# ⚠️ Common Beginner Mistakes

❌ Using `=` instead of `==`

❌ Forgetting that `/` returns a decimal value

❌ Mixing text (`str`) with numbers (`int`)
