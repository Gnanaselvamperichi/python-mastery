# 📖 What is User Input?

**User Input** means accepting information from the user while the program is running.

Instead of using fixed values, the program asks the user to enter data.

### 🌍 Real-Life Example

Think about an ATM 🏧

```
Enter PIN
     │
     ▼
Verify PIN
     │
     ▼
Show Account Balance
```

The ATM first **takes input**, then **processes it**, and finally **shows the output**.

Python works the same way!

---

# 💻 input() Function

The `input()` function is used to receive information from the user.

```python
name = input("Enter your name: ")
```

📌 **Note:** `input()` always returns data as a **String (`str`)**.

If you need numbers, convert them using:

```python
age = int(input("Enter your age: "))
```

---

# 🎨 How Input & Output Work
